Installation
------------

Install via pip (once published):

.. code-block:: bash

   pip install pykal

Or clone directly for development:

.. code-block:: bash

   git clone https://github.com/nehalsinghmangat/pykal.git
   cd pykal
   pip install -e ".[dev]"
