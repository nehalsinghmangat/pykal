pykal: A Kalman Filter Framework
================================

.. epigraph::

   " Prediction is very difficult, especially if it’s about the future."

   — Neils Bohr
   
`pykal` is a Python package for state estimation built around the Kalman Filter. It offers a unified framework for simulating dynamical systems and rapidly developing Kalman Filter implementations, ranging from the standard Extended Kalman Filter (EKF) to more advanced approaches such as Partial-Update and Observability-Aware filters.


The goal: **Make state estimation readable, easily debuggable, and extensible.**
