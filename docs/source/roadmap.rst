Project Roadmap
===============

This roadmap outlines planned features and development milestones for `pykal`.

Short-Term Goals
----------------

- ✅ Robust `System` interface with validation for continuous/discrete models
- ✅ Core filters: EKF, UKF, Information Filter (IF), Square-Root variants
- ✅ Partial-update Kalman filters (PKF, OPSKF) with dynamic β
- ✅ Simulation utilities with noise injection and reproducibility support

Mid-Term Goals
--------------

- 🔲 Add ensemble-based filters (EnKF, PEnKF)
- 🔲 Integrate dynamic observability analysis tools
- 🔲 Support for control design modules (e.g., LQR, MPC integration)
- 🔲 PyTorch & TensorFlow integration for hybrid model learning
- 🔲 Export-ready visual diagnostics (e.g., observability heatmaps)

Long-Term Vision
----------------

- 🌐 Web-based interactive demo notebooks
- 📦 Conda / PyPI packaging with CLI + API parity
- 🧪 Benchmark suite and comparative evaluations
- 📘 Publication-ready documentation and citation support
